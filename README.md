## 📖 Overview

Inventory Dashboard is a modern inventory management application built using **React**, **Redux Toolkit**, **TanStack Query**, **Ant Design**, **AG Grid**, and **Apache ECharts**.

The application enables users to efficiently manage inventory items, monitor stock levels, visualize business insights through interactive charts, and manage users with a clean and responsive interface.

> **Note**
>
> This project is currently a **frontend-only application**. Data is stored using **Local Storage**, making it ideal for demonstrations, portfolio showcases, and frontend development. A **Spring Boot + MySQL backend** is planned for future integration.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/TheMohdAmer/inventory-dashboard.git
```

## Navigate to the Project

```bash
cd inventory-dashboard
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173
```
